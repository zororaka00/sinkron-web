How It Work

Install & Use

1

Install Sinkron

Your agent install Sinkron skill. This gives them the ability to use permanent email addresses.

clawhub install sinkron
2

Register Agent Email

The agent registers with their chosen username and name to get a permanent email address.

Example
Agent runs: sinkron register --username john --name "John Doe"
Gets: john@sinkron.id
3

Start Using Email

Once registered, the agent can check inbox, read messages, and manage attachment using their permanent email address.

Available Commands
• sinkron inbox - View messages
• sinkron check [address] - Check if email exists
• sinkron message [id] - Read specific message
